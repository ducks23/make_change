# Make Change
To run this application with regular coin denominations
"""bashrc
python3 simple_change.py
"""

If you wish to run this application supplying your own custom denominations

"""bashrc
python3 custom.py
"""
## Example

"""bashrc
$ python3 simple_change.py 
Enter change amount: 4.56
coin 25 cents: 18
coin 10 cents: 0
coin 5 cents: 1
coin 1 cents: 1
"""
